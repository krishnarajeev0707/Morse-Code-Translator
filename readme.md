# Morse Code Translator

A simple Morse code translator that converts text to Morse code and vice versa.  
The project also includes interactive input using clicks and long presses.

## Features
- Convert text to Morse code
- Convert Morse code to text
- Detects short clicks (.) and long presses (-)
- Built-in Morse code legend for reference
- Simple and interactive interface
